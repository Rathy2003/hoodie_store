## Technologies
- PHP
- MySQL
- jQuery

## Features
- User registration and login
- Product display and purchase
- Order history
- Admin dashboard

## How to Use
1. Clone the repository
2. Import the database
3. Configure the database connection in ```.env``` file
4. Run the application
